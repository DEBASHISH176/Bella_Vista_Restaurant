# 🍽️ Bella Vista - Restaurant Website

A sleek, responsive, and visually stunning frontend landing page for **Bella Vista**, a fictional high-end restaurant. This project focuses on modern web design principles using pure HTML and CSS.

---

## 🌟 Project Overview
**Bella Vista** (Italian for "Beautiful View") is designed to provide a seamless digital experience for food lovers. The goal was to create a digital menu and reservation gateway that feels as elegant as the physical dining experience itself.

## 🚀 Features
- **Responsive Design:** Fully functional on mobile, tablet, and desktop views.
- **Hero Video/Image:** An immersive entrance that captures the restaurant's ambiance.
- **Digital Menu:** A beautifully categorized layout for Starters, Mains, and Desserts.
- **Interactive Elements:** Smooth CSS hover effects and transitions on buttons and navigation.
- **Clean Code:** Semantic HTML5 for better SEO and accessibility.

## 🛠️ Technologies Used
- **HTML5:** For structured, semantic content.
- **CSS3:** Custom styling, Flexbox/Grid layouts, and animations.
- **Google Fonts:** For sophisticated typography.
- **Font Awesome:** (Optional) For social media and contact icons.
